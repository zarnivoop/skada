# Skada Damage Meter

Skada is a modular damage meter for World of Warcraft with various viewing modes, segmented fights and customizable windows. It aims to be highly efficient with memory and CPU.

"Skada" is Swedish for "Damage".

## Features

- Multiple viewing modes (Bar, Inline, Data text)
- Segmented fights with saved history
- Customizable windows and appearance
- Support for LDB display addons (Titan Panel, ChocolateBar, ElvUI)
- Low memory and CPU usage
- Easy to extend with plugins
